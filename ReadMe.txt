# SecurePasswordTransfer

## Overview
This python project contains some simple applications that enable secure transfer of passwords over a TCP/IP network. It provides a secure way to transmit sensitive information such as passwords between two clients.

write_text_to_file writes input text to a .txt file.

password_generator randomly generates 50 unique passwords.

password_picker chooses 3 random passwords from a list of multiple passwords.

encrypt uses passwords to encrypt .txt files.

network_sender is able to send files between two pc's on the same network.

network_reciever is necesarry to recieve files send by network_sender

decryptor is necesarry to decrypt the files which have been encrypted by encrypt.

convert_to_pdf converts .txt files to .pdf files.
